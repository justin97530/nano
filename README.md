# Nano
I may have a sweet roll in my arm, but i'm totally not a robot.

### Installation
##### Prerequisites
- [NodeJS v0.10.2 +](https://nodejs.org/download/)

##### Installation
1. `cd ~/.../eew`
2. `git clone git@github.com:lumios/eew.git`
3. `sudo npm install`

##### Run
1. `cd ~/.../eew`
2. `npm start`
